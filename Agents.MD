## Агент анонимизации презентаций (Google Slides)

Цель: подготовить Google Apps Script для Google Slides, который автоматически анонимизирует презентацию перед отправкой/публикацией.

Скрипт будет запускаться на активной презентации и по шагам выполнять серии операций по «очистке» слайдов.

### Основные задачи (этапы)

1. **Удаление всех комментариев**
   - Пройти по всем слайдам текущей презентации.
   - Удалить все комментарии и ветки комментариев, чтобы в файле не оставалось обсуждений и служебной переписки.

2. **Удаление объектов вне видимой области слайда**
   - Рассматривать каждый слайд как прямоугольную видимую область (page bounds).
   - Для каждого объекта на слайде (фигуры, текстовые блоки, картинки и любые другие элементы):
     - Определить его ограничивающий прямоугольник (bounding box).
     - Проверить пересечение bounding box с видимой областью слайда.
   - **Правило:** 
     - Если объект **вообще не пересекается** с видимой областью, он **должен быть удалён**.
     - Если объект **хотя бы минимально заходит** в видимую область, он **должен быть сохранён** (не удаляем).
   - Таким образом удаляются все «запаркованные» за пределами слайда элементы (черновики, заготовки, скрытые подсказки и т.п.).

3. **Анонимизация содержимого (будущий этап)**
   - Логика замены/обфускации реальных имён, названий компаний, логотипов, метаданных и т.д.
   - Правила анонимизации (что именно и как заменять) будут определены отдельно и реализованы в виде отдельных функций/агентов.

### Технические заметки

- **Среда**: Google Apps Script для Google Slides.
- **Объект работы**: активная презентация (файл, в котором запускается скрипт).
- **Уровень воздействия**: скрипт модифицирует саму презентацию (это не визуальный фильтр, а именно изменение документа).
- Скрипты будут разбиты на логические части:
  - функции для работы с комментариями;
  - функции для анализа геометрии объектов и проверки их нахождения внутри/вне видимой зоны;
  - функции для анонимизации содержимого (будут определены позже).
